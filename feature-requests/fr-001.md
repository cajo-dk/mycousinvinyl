# Feature Request

- REQUEST ID: FR-001
- DATTE: 2025.12.21

## Abstract

The purpose of this FR is to make it easier for the user to add records to their collection, if the album and pressing are already in the systm.

## User Stories

### Story #1

- Required access level: Editor or Admin

As a user, when viewing the Album List, I want to be able to click a button in the album row to add a pressing for the album with a check mark option (default: checked) to add the pressing to my collection.

### Story #2

- Required access level: Editor or Admin

As a user, when viewing an album's detail page, I want a list of known pressings shown at the bottom of the album's details, with each line having a button to add the pressing to my collection. The list itself should provide at button for adding a pressing.
