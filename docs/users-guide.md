# User's Guide

## Purpose

This guide helps end users understand the core screens and complete typical tasks.

## 1. Walk-Through

### Artists

- Browse artists in a table view.
- Search and filter by name.
- Open an artist to view albums and related details.
- Create or edit artists (Editor/Admin).

### Albums

- Browse albums and filter by artist or title.
- View album details and related pressings.
- Create albums manually or via Discogs lookup (Editor/Admin).

### Pressings

- Browse pressings across albums.
- Add new pressings from Discogs releases.
- Edit pressing details (format, size, speed, edition, barcode).

### Collection Items

- Manage your owned pressings.
- Track condition, purchase info, play counts, and notes.
- Use table actions to view, edit, or remove items.

### Statistics

- The Home page summarizes collection totals and value estimates.
- See latest additions and purchase summary.

### Profile

- Update display preferences.
- Configure Discogs OAuth or Personal Access Token.
- Select profile icon and colors for sharing views.

## 2. Creating the first item in your collection

1. Go to **Artists** and create or select the artist.
2. Go to **Albums** and create an album for that artist.
3. Go to **Pressings** and create a pressing.
   - If available, use Discogs search to auto-fill details.
4. Go to **Collection** and add the pressing to your collection.
5. Set media/sleeve condition, purchase price, and any notes.

After this, the item appears in your collection and counts toward stats on the Home page.
